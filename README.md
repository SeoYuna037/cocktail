# Let's Drink!
### 프로젝트 소개
Let's Drink는 CocktailDB API를 활용하여 원하는 칵테일의 이름을 검색하고 자세한 정보를 확인할 수 있는 사이트입니다.


### 사용 기술
-React, Scss
